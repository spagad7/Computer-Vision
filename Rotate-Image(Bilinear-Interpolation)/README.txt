Rotate-Image
============

rotateImg.m
===========
This is a matlab code to rotate an image about a point in a plane and perform bilinear-interpolation to plug holes in rotated image. 

Arguments
=========
Arg1:   gray-scale image to rotate
Arg2:   angle of rotation in degrees
Arg3:   point about which to rotate image [0,0] is the top left corner of the input image.
